# 🚀 DevOps & Quality Assurance - FECAP

Este repositório centraliza os estudos, exercícios e projetos práticos da disciplina de **DevOps**, com foco em **Qualidade de Software (QA)**, automação de testes e cultura de entrega contínua. 

Como aluno de **Análise e Desenvolvimento de Sistemas (ADS)** na FECAP, este material reflete a transição de testes manuais para rotinas automatizadas e testes de carga.

---

## 📌 Conteúdo das Aulas

### 1. Automação de Testes com Python (`2026_teste_qualidadeDevops.ipynb`)
Este módulo aborda a evolução da validação de software:
* **Testes de Unidade:** Criação de funções de soma e rotinas de validação de resultados esperados.
* **Testes de Caixa Preta:** Implementação de testes de exaustão utilizando a biblioteca `random` para gerar massas de dados aleatórias.
* **Testes via Arquivo:** Automação de baterias de testes através da leitura de arquivos `.txt` (Massa de Dados), permitindo validar grandes volumes de registros de uma só vez.

### 2. Validação de Regras de Negócio (`Untitled0.ipynb`)
Focado na aplicação de lógica de QA em cenários de avaliação:
* **Lógica de Condicionais:** Desenvolvimento de funções para validar notas e atribuir conceitos.
* **Verificação de Robustez:** Comparação sistemática entre o `resultado_real` e o `resultado_esperado` com logs visuais de sucesso (✅) ou falha (❌).

### 3. Infraestrutura e Automação de Logs
* **Shell Scripting:** Uso de comandos Linux (`find`, `grep`, `head`, `tail`) para automação de análise de logs e geração de relatórios semanais.
